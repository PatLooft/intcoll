# intcoll
various uses of data structures via the same program
The program uses 3 different lists: a positive list (P), a negative list (N), and a list in which the last instance of the number was positive (L).
Example input:
5
4
3
2
1
-4
-3
-2
4
0

OUTPUT:
P: 5, 4, 3, 2, 1
N: -4, -3, -2
L: 5, 1, 4
